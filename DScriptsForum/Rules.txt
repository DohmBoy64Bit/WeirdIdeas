Project Vision

Goal:
Build a secure, modular, and professional forum + script hub for Roblox scripts using Python (FastAPI or Flask) + TailwindCSS, with:

Full forum features (threads, posts, user CP, rep, ranks, awards)

Script hub homepage like rscripts.net (recent, trending, likes, downloads)

Admin validation system

Responsive mobile-first UI

Clean separation of concerns (PEP + DREY principles)

1. Technology Decisions
Backend

Framework: FastAPI (preferred for async, validation, speed)
Flask acceptable if simplicity is prioritized.

Database: PostgreSQL (primary) + Redis (caching, sessions, rate limiting)

ORM: SQLAlchemy 2.0 + Alembic migrations

Auth:

Argon2 password hashing

JWT + HttpOnly cookies

One-time registration codes

File storage: Local or S3-compatible

Queue: Celery or RQ (emails, processing)

Frontend

TailwindCSS

Jinja2 templates (SSR first)

Alpine.js for light interactivity

No heavy SPA frameworks initially

Security Stack

CSRF via tokens

Rate limiting

Input validation with Pydantic

Content sanitization (Bleach)

CSP headers

Download scanning placeholders
